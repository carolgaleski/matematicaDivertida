# Matemática Divertida

**Descrição:**  
O projeto **Matemática Divertida** é um aplicativo educativo desenvolvido para a disciplina **Desenvolvimento para Dispositivos Móveis**, utilizando **Kotlin** e **Android Studio**. O app tem como objetivo tornar o aprendizado da matemática mais divertido e interativo voltado para crianças.

---

## Funcionalidades

- Resolver operações matemáticas (adição e subtração, contagem de números, resolução do maior número)  
- Quizzes interativos com pontuação  
- Feedback imediato para respostas corretas e incorretas  
- Interface simples e intuitiva, ideal para crianças 

---

## Tecnologias Utilizadas

- **Linguagem:** Kotlin  
- **IDE:** Android Studio  
- **Plataforma:** Android  
- **Recursos:** Layouts XML, Activities, Intents e lógica de manipulação de dados
